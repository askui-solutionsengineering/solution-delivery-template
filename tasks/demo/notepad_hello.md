# Notepad Hello Task

## Objective
Open Notepad (or the default text editor on your system), type a specific message, save the file, and report completion.

## Steps

1. **Open Notepad** (or the default text editor on your system).

2. **Type exactly:** `"Hello from AskUI Agent - task completed at [current date/time]"`.

3. **Save the file** to the agent workspace with the name `notepad_hello_output.txt`.

4. **Close Notepad** and report the saved file path and a short confirmation in your task summary. Include a screenshot of the saved file or the editor before closing.

## Deliverables
- Saved file: `notepad_hello_output.txt` in the agent workspace
- Task summary with saved file path and confirmation
- Screenshot of the saved file or the editor before closing
